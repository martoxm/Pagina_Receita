# Página de Receita — Treino de HTML e CSS

Este repositório é um projeto de treino para praticar HTML e CSS. Contém uma página simples com a receita de "Cupcake de café com chantilly" para exercitar marcação semântica e estilos.

## Como visualizar

- Abra o arquivo [index.html](index.html) em um navegador.
- Ou use uma extensão como Live Server no VS Code para ver alterações em tempo real.

## Estrutura do projeto

- `index.html` — página principal com conteúdo da receita
- `style.css` — arquivo de estilos
- `assets/` — imagens e outros recursos

## Tecnologias

- HTML5
- CSS3
- Google Fonts
